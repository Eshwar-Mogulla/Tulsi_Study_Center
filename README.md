# Tulsi Study Center Website

A premium, modern, and professional website built for "Tulsi Study Center (Library)". The site is designed to look corporate, minimal, structured, and trustworthy, offering a premium co-working/study space brand experience.

![Tulsi Study Center Logo](./public/favicon.svg)

## 🌟 Features

- **Hero Section**: Premium first impression with a clear value proposition, subtle fade-in, and slide-up animations.
- **Facilities Grid**: Modern responsive grid layout highlighting comfortable seats, CCTV, high-speed WiFi, black coffee, and more, complete with soft shadows and hover reveals.
- **Open Hours Banner**: Dark-themed highlight section with soft glow accent borders.
- **Dynamic Gallery**: WebP-optimized, responsive masonry-style image grid showing the premium interior with subtle hover zoom effects.
- **Registration CTA**: Visually strong call-to-action block with a gentle pulsing glow effect.
- **Contact & Map**: Clean contact details paired with a responsive Google Maps embed.

## ⚙️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS V4
- **Animations**: Framer Motion
- **Icons**: React Icons (FontAwesome pack)

## 🎨 Design System

**Typography:**

## 🚀 Getting Started

### Prerequisites

- Node.js installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/eshwarla/Tulsi_Study_Center.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Tulsi_Study_Center
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

Start the local Vite development server:
```bash
npm run dev
```

The application will typically be available at `http://localhost:5173`.

### Building for Production

To create a production-ready build:
```bash
npm run build
```

The built files will be located in the `dist` directory. You can preview the production build locally using:
```bash
npm run preview
```
